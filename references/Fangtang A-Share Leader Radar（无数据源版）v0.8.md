## 定位

方塘追龙用于在A股阶段性主线中识别、比较和分析龙头候选、前排候选、中军候选、补涨候选和异动标的。

本框架不只用于选股，也用于单个股票分析、多个股票横向比较、股票池筛选和全市场主线筛选。输出重点是“市场是否支持”“主线是否成立”“个股是否属于主线结构”“个股四个面是否共振”“候选层级和置信度如何”，不输出具体执行计划。

## 适用场景

- 单个股票分析：判断一只股票所处市场、所属主线、板块角色、基本面、故事面、动量面和资金面。
- 多个股票对比：统一口径比较多个候选，判断谁更像龙头、前排、中军、补涨或异动。
- 股票池筛选：对一批股票先做初筛，再对入池股票做详细分析和分层。
- 全市场筛选：先寻找当前强主线，再从主线中筛选高辨识度个股。
- 板块复盘：判断一个行业或题材是否形成主线，以及龙头、中军、前排、补涨结构是否清晰。

## 核心原则

- 先识别任务，再确定数据口径；不要用股票池筛选口径套单股分析。
- 单股分析也必须判断市场环境和板块强度，不能只看个股自身走势。
- 市场环境是约束项，主线板块和个股动量是核心项，基本面、故事面、资金面是验证项。
- 龙头不是单纯涨幅最大，而是“主线地位、板块认同、成交聚焦、相对强度、故事辨识度、趋势结构”共同确认。
- A股追龙必须加入赚钱效应、涨停生态、连板高度、板块扩散、政策题材和风格偏好。
- 基本面用于判断行情支撑力和持续性，不机械套用美股CAN SLIM。
- 数据缺失时降低置信度，不编造字段，不把单一指标当成最终结论。

## 分析流程

```
0. 任务识别与数据口径
1. 市场环境分析
2. 主线板块分析
3. 个股初步筛选 / 个股角色初判
4. 个股详细分析
   4.1 基本面
   4.2 故事面
   4.3 动量面
   4.4 资金面
5. 横向比较与候选分层
6. 信息缺口与置信度
```

市场环境决定分析结论的上限，主线板块决定“龙”是否有土壤，个股初筛或角色初判决定是否值得展开，四个面用于解释和验证，最后通过横向比较和置信度形成候选分层。

## 0. 任务识别与数据口径

### 目标

先识别用户任务属于哪一类，再决定分析深度、比较对象和输出方式。

### 任务模式

| 模式 | 适用输入 | 执行重点 | 输出重点 |
|------|----------|----------|----------|
| 单个股票分析 | 1只股票代码或名称 | 完整分析市场、所属主线、个股角色和四个面 | 板块角色、龙头确认度、四面强弱、信息缺口 |
| 多个股票对比 | 2只以上股票 | 先比较所属主线，再比较个股四个面 | 横向排名、强弱差异、候选层级 |
| 股票池筛选 | 一批股票、行业成分股、题材池 | 先批量初筛，再对入池候选详细分析 | 入池名单、剔除理由、分层结果 |
| 全市场筛选 | 未指定股票 | 先找强主线，再从主线中筛前排个股 | 当前主线、候选表、后续验证清单 |
| 板块复盘 | 一个行业、概念或题材 | 判断主线强度和内部结构 | 龙头、中军、前排、补涨分布 |

### 执行差异

- 单个股票分析：第3步改为“个股角色初判”，不是判断是否从股票池剔除，而是判断它在所属板块中的位置。
- 多个股票对比：如果候选不在同一主线，必须先比较主线强度，再比较个股本身。
- 股票池筛选：第3步承担批量过滤作用，第4步只对入池或观察候选展开。
- 全市场筛选：第2步权重最高，先找到主线，再筛个股。

### 数据口径

每次输出必须先说明：

- 数据日期：行情、财务、公告、事件分别截至哪一天。
- 数据来源：通达信、同花顺、万得、东方财富、交易所公告、公司公告或CSV/XLSX文件。
- 数据范围：全市场、某板块、用户给定股票池，还是指定股票。
- 数据缺口：哪些关键数据没有取得，哪些指标使用代理变量。

## 1. 市场环境分析

### 目标

判断当前A股市场是否支持强势主线和龙头股表现。

市场环境不是普通加分项，而是候选层级的约束项。弱市场中也可能出现强股，但结论应标明“逆势强”或“局部强”，不能直接等同于进攻环境中的主线龙头。

### 核心问题

- 当前市场是进攻、中性还是防守？
- 赚钱效应是否真实存在，还是只有指数上涨？
- 当前风格是否偏向追龙策略适用的方向？
- 短线生态是否支持强势股持续表现？

### 关注指标

| 模块 | 观察内容 |
|------|----------|
| 指数趋势 | 沪深300、中证500、中证1000、创业板指、科创50、全A指数的趋势和均线位置 |
| 成交活跃度 | 全市场成交额、成交量变化、放量或缩量状态 |
| 赚钱效应 | 上涨家数、下跌家数、阶段新高数量、阶段新低数量 |
| 短线生态 | 涨停家数、跌停家数、连板高度、炸板率 |
| 风格偏好 | 大盘/小盘、成长/价值、题材/权重 |
| 辅助资金 | 两融余额、ETF净申购、北向资金，如数据可得 |

### 状态划分

| 状态 | 特征 | 分析约束 |
|------|------|----------|
| 进攻 | 指数趋势向上，成交放大，涨停和连板活跃，赚钱效应较好 | 可正常寻找龙头和前排候选 |
| 中性 | 指数震荡，成交一般，板块轮动较快，赚钱效应不稳定 | 提高确定性要求，优先主线前排和中军 |
| 防守 | 指数走弱，成交萎缩，跌停或炸板增加，持续性差 | 候选层级整体下调，重点识别逆势强或局部强 |

### 分析规则

- 指数强但上涨家数少、涨停少、连板弱，不视为完整进攻环境。
- 成交额明显萎缩时，题材持续性和板块扩散能力要降权。
- 风格偏向权重防御时，小票题材龙头的评价要更谨慎。
- 市场环境弱时，不否定强股，但降低结论置信度和候选层级。

## 2. 主线板块分析

### 目标

确认市场资金和情绪是否集中在某个行业、产业链或题材方向，并判断该方向是否具备主线特征。

主线板块是追龙框架的核心。A股多数龙头不是孤立产生，而是在政策、产业、题材、资金和情绪共振中形成。

### 核心问题

- 当前主线是什么？
- 主线属于行业景气、政策催化、产业趋势、事件驱动，还是短期情绪轮动？
- 板块内部是否形成“龙头 + 中军 + 前排 + 补涨”的结构？
- 板块处在启动、确认、扩散、分化还是退潮阶段？

### 主线类型

| 类型 | 观察重点 |
|------|----------|
| 行业主线 | 行业景气、产品价格、订单、产能、供需结构、行业指数强度 |
| 概念题材 | 政策事件、技术突破、产业趋势、市场关注度、题材扩散 |
| 产业链主线 | 上游材料、中游设备、下游应用之间是否联动 |
| 风格主线 | 大盘/小盘、成长/价值、题材/权重的阶段性偏好 |

### 主线阶段

| 阶段 | 特征 | 分析重点 |
|------|------|----------|
| 启动 | 少数前排率先走强，板块刚开始放量 | 识别率先涨停、率先突破、成交聚焦个股 |
| 确认 | 板块指数走强，涨停扩散，中军稳住 | 确认龙头、中军和前排结构 |
| 扩散 | 同题材、产业链上下游、补涨方向扩散 | 区分健康扩散和后排乱涨 |
| 分化 | 强者继续强，后排掉队，炸板率上升 | 只保留龙头、中军和最强前排 |
| 退潮 | 龙头走弱，板块成交萎缩，亏钱效应扩散 | 降低候选层级，转入复盘和跟踪 |

### 关注指标

| 模块 | 观察内容 |
|------|----------|
| 相对强度 | 行业或题材相对强度是否进入全市场前20%-30% |
| 指数表现 | 板块指数是否强于全A和主要宽基指数 |
| 成交聚焦 | 板块成交额是否放大，是否连续获得资金关注 |
| 涨停扩散 | 板块内涨停家数、连板高度、阶段新高数量 |
| 板块宽度 | 成分股站上50日、200日均线比例是否提升 |
| 催化来源 | 政策、产业周期、产品涨价、订单、技术突破、事件驱动 |
| 内部结构 | 是否形成龙头、中军、前排、补涨的层次 |

### 主线强度判断

| 强度 | 特征 |
|------|------|
| 强主线 | 板块指数领先、成交放大、涨停扩散、前排清晰、中军稳住、题材有持续催化 |
| 次主线 | 板块表现较强，但扩散、成交或催化持续性不足 |
| 轮动题材 | 短期有表现，但板块结构松散，持续性需要继续观察 |
| 异动方向 | 单一个股或少数个股上涨，尚未形成板块共振 |
| 非主线 | 板块强度弱，个股强度主要来自孤立因素 |

### 分析规则

- 单一股票上涨不等于板块强势，至少要看到板块内多只个股共振。
- 主线刚启动时，重点看率先涨停、率先突破和成交聚焦个股。
- 主线中后段，重点看分化后仍保持强度的龙头和中军。
- 板块过热、扩散失控、炸板率升高时，降低后排补涨层级。

## 3. 个股初步筛选 / 个股角色初判

### 目标

根据任务模式，判断个股是否值得详细分析，或判断单只股票在所属板块中的角色。

### 模式差异

| 任务模式 | 本步骤名称 | 处理方式 |
|----------|------------|----------|
| 单个股票分析 | 个股角色初判 | 不做剔除式筛选，重点判断它在板块中是什么角色 |
| 多个股票对比 | 个股初步筛选 + 角色初判 | 统一比较强度、成交、相对位置和板块角色 |
| 股票池筛选 | 个股初步筛选 | 先过滤缺乏主线、强度或成交支撑的标的 |
| 全市场筛选 | 主线内初筛 | 只从强主线和次主线中筛前排、中军和补涨 |

### 初筛指标

| 模块 | 观察内容 |
|------|----------|
| 涨幅排名 | 近5日、10日、20日涨幅在板块中的位置，优先前5名或前10% |
| 成交排名 | 成交额和换手率是否处于板块前列 |
| 相对强度 | 是否强于所属板块、全A指数和主要宽基指数 |
| 抗跌修复 | 市场调整时是否更抗跌，市场修复时是否率先走强 |
| 率先表现 | 是否率先涨停、连续涨停、率先突破或平台新高 |
| 市值弹性 | 流通市值是否既有弹性又能承载资金，常见弹性区间为50-500亿 |
| 指标失真 | 是否存在连续一字、停牌复牌、极端缩量导致强度失真 |

### 角色初判

| 角色 | 典型特征 | 分析重点 |
|------|----------|----------|
| 情绪龙头 | 率先涨停或连板，带动板块情绪，弹性高 | 连板高度、涨停号召力、板块带动性 |
| 趋势龙头 | 沿均线稳步上行，成交持续活跃，中线资金认可度较高 | 相对强度、趋势结构、基本面和故事支撑 |
| 中军 | 市值较大，成交额高，对板块指数影响大 | 成交额、板块稳定作用、机构关注 |
| 前排跟随 | 涨幅和成交靠前，但独立带动性弱 | 与龙头差距、持续性、补涨空间 |
| 补涨候选 | 逻辑相关、位置较低，启动晚于龙头 | 板块扩散阶段、基本面匹配度 |
| 异动股 | 短期放量大涨，但板块和逻辑支撑不足 | 异动原因、是否能纳入主线结构 |
| 非主线标的 | 个股自身有表现，但所属方向不是当前主线 | 是否只是独立事件或个股基本面修复 |

### 初筛结论

| 判断 | 标准 |
|------|------|
| 进入详细分析 | 市场或主线较强，个股强度、成交、相对表现均靠前 |
| 观察候选 | 板块或个股有亮点，但强度、成交或趋势仍需确认 |
| 主题跟踪 | 符合题材方向，但个股地位和成交聚焦度不足 |
| 非主线跟踪 | 个股有独立表现，但缺乏板块结构支持 |
| 剔除 | 缺乏主线、强度或成交支撑 |

## 4. 个股详细分析

### 目标

对入池个股或指定股票做结构化分析，判断其龙头确认度、研究优先级、候选层级和置信度。

个股详细分析分为四个面：基本面、故事面、动量面、资金面。

## 4.1 基本面

### 目标

判断公司质地、盈利质量和业务逻辑能否支撑市场对它的关注。

### 先识别逻辑类型

| 类型 | 关注重点 | 易错点 |
|------|----------|--------|
| 成长型 | 收入、利润、ROE、现金流、订单和产能持续增长 | 单季度爆发被误判为长期成长 |
| 周期型 | 价格周期、库存周期、产能周期、吨利润、利润拐点 | 周期高点低PE被误判为便宜 |
| 困境反转型 | 亏损收窄、订单改善、资产重组、政策改善、现金流修复 | 反转证据不足，仅靠预期驱动 |
| 主题题材型 | 政策催化、产业趋势、技术突破、事件密度、市场关注度 | 概念相关度弱，缺乏业绩映射 |
| 资产重估型 | 资产注入、并购重组、资源重估、业务结构变化 | 信息不透明，预期兑现难度高 |

### 检查指标

- 近2-3年营收、归母净利润、扣非净利润趋势。
- 最近季度收入和利润增速是否同步。
- ROE、毛利率、净利率是否稳定或改善。
- 经营现金流是否支持利润质量。
- 应收账款、存货、商誉、资产负债率是否出现明显异常。
- PE、PB、PS、PEG等估值指标是否能被行业地位、成长性或周期位置解释。
- 机构一致预期或研报关注是否能解释估值和上涨逻辑，如数据可得。

### 分析规则

- 基本面强，不自动等于龙头；还要看市场和板块是否认可。
- 题材强但基本面弱，可以作为主题观察，不应直接归为高质量龙头。
- 周期股要先判断周期位置，再看估值和利润增速。
- 困境反转股必须说明反转证据是已经出现，还是仍停留在预期阶段。
- 基本面数据缺失时，不要用“质地好/差”做强结论，只能写“基本面待验证”。

## 4.2 故事面

### 目标

判断市场正在交易什么叙事，以及叙事是否清晰、可验证、可扩散。

### 关注内容

| 模块 | 观察内容 |
|------|----------|
| 核心标签 | 产品放量、订单爆发、涨价、国产替代、政策扶持、技术突破、AI赋能、资产重组等 |
| 业务映射 | 公司业务是否真实对应题材，收入、客户、产品、订单是否能支撑标签 |
| 事件密度 | 财报、业绩预告、重大合同、产能投产、产品发布、行业政策、投资者交流 |
| 可信度 | 是否有合同金额、客户名称、产能数据、公告或财报数据可验证 |
| 扩散能力 | 是否从单一股票扩散到产业链、同题材、上下游或可比公司 |
| 市场记忆 | 是否有历史辨识度、历史龙头记忆、反复活跃的题材标签 |

### 证据等级

| 等级 | 标准 | 结论约束 |
|------|------|----------|
| A级 | 有公告、财报、合同、客户、产能或价格数据可验证 | 可作为强支撑 |
| B级 | 逻辑自洽，业务能对应，但关键数据尚未完全兑现 | 可作为中等支撑 |
| C级 | 主要依赖概念标签、市场想象或社交讨论 | 只能作为弱支撑 |

### 分析规则

- A股强势股常由政策、事件和题材驱动，但持续性仍要回到板块强度和成交聚焦。
- 故事越清晰、越容易被市场复述，越有利于提高辨识度。
- 只有概念标签、没有业务映射或数据验证时，应降低分析置信度。
- 同一故事下，优先分析业务映射更直接、成交更聚焦、板块角色更明确的标的。

## 4.3 动量面

### 目标

判断个股是否具备龙头应有的价格强度、趋势结构和相对表现。

动量面是追龙框架中最重要的个股维度。没有动量支撑的个股，即使基本面和故事面较好，也不应直接归为龙头候选。

### 龙头地位指标

- 近5日、10日、20日涨幅在板块中的位置。
- 成交额和换手率是否处于板块前列。
- 是否强于所属板块、全A指数和主要宽基指数。
- 市场调整时是否更抗跌，市场修复时是否率先走强。
- 是否率先涨停、连续涨停，或涨停后带动同板块扩散。
- 个股上涨时板块是否跟随，个股走弱时板块是否明显降温。
- 与第二名差距是否明显，还是仅为同质化前排之一。

### 趋势结构

日线结构：

- 股价是否站上20日、60日、120日、150日、200日均线。
- 20日、60日均线是否向上，短中期趋势是否共振。
- 股价距离52周高点是否不超过25%。
- 股价较52周低点是否已经上涨25%以上。
- 突破时是否放量，回调时是否缩量。
- 涨跌停、连续一字板、停牌复牌是否导致趋势指标失真。

周线结构：

- 周线MA10、MA30是否向上。
- 周线是否处于平台突破、主升、加速、震荡或退潮阶段。
- 日线强度是否得到周线结构确认。

相对强度：

- 全市场相对强度优先前20%，更优为前10%。
- 行业内优先排名前三或前10%。
- 股价表现应强于所属行业和主要宽基指数。
- 市场调整期抗跌、修复期领先的个股优先级更高。

## 4.4 资金面

### 目标

用成交和资金行为辅助判断市场认同度。

资金面只作为辅助证据，不替代主线强度、故事验证和动量强度。

### 关注指标

| 模块 | 观察内容 |
|------|----------|
| 成交额 | 是否连续位居板块前列，是否能够承载资金进出 |
| 量价关系 | 是否上涨放量、回调缩量，是否出现放量滞涨 |
| 换手结构 | 换手率是否与股性、市值和阶段位置匹配 |
| 主力资金 | 主力资金、超大单、大单等数据，如数据可得 |
| 北向和融资 | 北向持仓、北向流向、融资余额变化，如数据可得 |
| 龙虎榜 | 机构席位、游资席位、买卖集中度，如数据可得 |
| 持仓线索 | 基金持仓、股东户数、户均持股、回购或管理层持股，如数据可得 |

### 分析规则

- 成交额和相对强度通常比单日资金净流入更重要。
- 资金流数据口径差异较大，只作为辅助线索。
- 资金关注与板块扩散同步时，龙头确认度更高。
- 高成交额不必然等于强势，还要结合涨幅、换手和板块位置。

## 5. 横向比较与候选分层

### 目标

把单个股票的分析结论放回市场和板块结构中比较，形成候选层级。

### 优先级逻辑

追龙优先级不按简单平均分决定，应按以下顺序判断：

```
市场环境约束
-> 主线强度
-> 个股角色和动量
-> 故事面验证
-> 基本面支撑
-> 资金面辅助
```

### 候选层级

| 层级 | 标准 |
|------|------|
| A级龙头候选 | 强主线或次强主线中，个股角色清晰，动量领先，故事面至少B级，成交聚焦，信息缺口可控 |
| B级前排候选 | 板块地位较强，但龙头确认度、故事验证或基本面支撑仍有短板 |
| C级观察候选 | 有主题、趋势或基本面亮点，但主线强度或个股角色不够明确 |
| D级主题跟踪 | 更适合跟踪板块演化，不宜列为核心候选 |
| E级剔除 | 缺乏主线、强度、成交或逻辑支撑 |

### 横向比较表

| 指标 | 候选A | 候选B | 候选C |
|------|-------|-------|-------|
| 所属主线 | | | |
| 主线强度 | | | |
| 板块阶段 | | | |
| 个股角色 | | | |
| 龙头确认度 | | | |
| 基本面 | | | |
| 故事面 | | | |
| 动量面 | | | |
| 资金面 | | | |
| 置信度 | | | |
| 候选层级 | | | |

### 辅助评分

评分只用于排序和分层辅助，不代表精确结论。

| 模块 | 属性 | 参考权重 | 说明 |
|------|------|----------|------|
| 市场环境 | 约束项 | 不直接决定排名 | 决定候选层级上限和置信度 |
| 主线板块 | 核心项 | 25 | 判断“龙”是否有板块土壤 |
| 角色与动量 | 核心项 | 30 | 判断个股是否真强、是否有辨识度 |
| 故事面 | 验证项 | 20 | 判断市场叙事是否清晰、可验证、可扩散 |
| 基本面 | 验证项 | 15 | 判断业绩和业务是否支撑行情持续性 |
| 资金面 | 辅助项 | 10 | 判断成交和资金是否认可 |

### 降级约束

- 市场环境为防守时，除非板块和个股明显逆势共振，否则最高为B级。
- 主线强度为轮动题材或异动方向时，最高为C级。
- 个股动量明显弱于板块时，不得评为A级。
- 故事面为C级且缺乏业务映射时，最高为C级。
- 基本面关键数据缺失时，不得给出高置信度结论。
- 关键数据缺失且无法替代验证时，整体候选层级下调一级。

## 6. 信息缺口与置信度

### 目标

明确哪些结论是数据支持的，哪些只是推断，避免分析看起来过度确定。

### 信息缺口分类

| 类型 | 示例 | 对结论的影响 |
|------|------|--------------|
| 行情缺口 | 缺少完整成交额、换手率、涨停/炸板、连板数据 | 降低动量面和资金面置信度 |
| 板块缺口 | 所属概念不清、板块成分不完整、产业链映射不明确 | 降低主线和角色判断置信度 |
| 财务缺口 | 缺少最近季度营收、扣非利润、现金流、ROE | 降低基本面置信度 |
| 事件缺口 | 催化剂未找到公告或权威来源 | 降低故事面证据等级 |
| 口径缺口 | 不同数据源行业分类、资金流口径不一致 | 标注口径差异，不强行合并 |

### 置信度分级

| 置信度 | 标准 |
|--------|------|
| 高 | 行情、板块、财务、事件数据较完整，主线和个股角色均可交叉验证 |
| 中 | 行情和板块数据完整，但财务或事件数据部分缺失 |
| 低 | 主要依赖行情或概念标签，缺少财务、公告或板块结构验证 |

---

## 数据需求

### 市场数据

- 主要指数日线行情、成交额、成交量、均线。
- 上涨家数、下跌家数、创新高和创新低数量。
- 涨停家数、跌停家数、连板高度、炸板率。
- 两融余额、ETF净申购、北向资金，如数据可得。

### 板块数据

- 申万、Wind、中信、通达信或同花顺行业分类。
- 行业指数涨跌幅、成交额、相对强度排名。
- 板块内涨停数量、阶段新高数量、前排个股涨幅排名。
- 政策事件、产业链标签、热点持续天数。
- 板块内“龙头 + 中军 + 前排 + 补涨”结构。

### 个股数据

- 股票代码、名称、交易所、行业、概念标签、流通市值。
- 前复权日线和周线行情，成交量、成交额、换手率。
- 20/50/60/120/150/200日均线，周线MA10/MA30。
- 近5日、10日、20日涨幅及板块排名。
- 52周高点、52周低点、全市场和行业相对强度。
- PE、PB、PS、PEG等适配估值指标。

### 基本面与事件数据

- 单季度收入、归母净利润、扣非归母净利润、EPS及同比。
- 最近8个季度收入和利润增速。
- 最近3-5年收入、利润、ROE、毛利率、净利率、经营现金流。
- 订单、产能、价格、库存、政策、产品发布、重大合同等催化信息。
- 机构关注、基金持仓、北向持仓、龙虎榜、股东户数等数据，如可得。

## 数据源建议

- 通达信：日线行情、均线、成交量、板块、公式初筛、行情快速扫描。
- 同花顺：概念题材、热点强度、资金流、事件标签、产业链标签、人气辅助指标。
- 万得：财务指标、一致预期、行业分类、指数、机构持仓、公告和事件数据。
- 东方财富：行情、板块、资金流、公告、互动和基础财务辅助数据。
- 交易所公告和公司公告：重大合同、重组、监管问询、财报披露等事件核验。

## 使用注意

- 不要只按涨幅排序，必须结合板块强度、成交聚焦、相对强度和逻辑支撑。

- 不要把概念相关等同于业务受益，需要核验收入、订单、客户或产品映射。
- 不要因为估值高直接否定强势龙头，也不要因为位置低就认定有补涨价值。
- 数据缺失时必须写入信息缺口，并降低候选层级或分析置信度。
